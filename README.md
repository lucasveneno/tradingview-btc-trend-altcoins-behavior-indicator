## Features ✨

- **BTC Trend Detection**
  - Mode 1: Price-based
  - Mode 2: EMA-based
- **Multi-Timeframe Support**
  - 1 min, 15 min, 1h, 4h, 1d, 1w, 1M
- **Altcoins Behavior Analysis**
  - Visualizes altcoins performance relative to BTC
- **Customizable Moving Averages**
  - Up to 4 EMAs + 1 SMA with adjustable values/colors
- **Summary Table**
  - Quick overview of trends and altcoin behavior
- **Visual Trend Lines**
  - Extend EMAs directly on chart

## Installation ⚡

1. Open [TradingView](https://www.tradingview.com/).
2. Go to **Pine Editor**.
3. Copy and paste the script code.
4. Click **Add to Chart**.

Alternatively, try the script directly on TradingView:  
👉 [Try the Indicator on TradingView](https://www.tradingview.com/script/jgwMdlN7/)

---

## Usage 💡

- Monitor BTC trend across multiple timeframes.
- Track altcoins relative to BTC dominance.
- Customize EMAs/SMA for personal strategy.
